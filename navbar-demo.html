Navbar Demo - Logged In State

This page shows how the navbar looks when a user is logged in.

Desktop View (≥768px)

  • User Avatar: Shows initials "RK" with gradient background
  • User Name: "Rahul Kumar" (visible on large screens)
  • Dropdown Arrow: Click to toggle the dropdown menu
  • Dropdown Menu: Contains user info, links to My Account, Orders, Change Password, and Logout button

Mobile View (<768px)

  • Click the hamburger menu (three lines) in the top right
  • The mobile menu will slide in from the left
  • Shows user header with avatar, name, and email
  • Contains all user menu options below the navigation links
Note: On the actual site, this logged-in state will only show when a user has successfully logged in through the signin page. The JavaScript automatically switches between logged-out and logged-in states based on localStorage.
Interactive Demo: Click the user avatar/name in the navbar above to see the dropdown menu open!
